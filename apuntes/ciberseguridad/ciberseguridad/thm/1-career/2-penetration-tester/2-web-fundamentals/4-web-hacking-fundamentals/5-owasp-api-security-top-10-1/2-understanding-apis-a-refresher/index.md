<h2>¿Qué es una API y Por Qué es Importante?</h2>
API significa Application Programming Interface. Es un medio que facilita la comunicación de dos componentes de software usando un conjunto de protocolos y definiciones. En el contexto de una API, una **Application** se refiere a cualquier software que tiene una funcionalidad específica e **Interface** se refiere a contrato entre dos apps que hace la comunicación posible mediante peticiones y respuestas. La documentación de cada API recoge cómo han estructurado dichas peticiones y respuestas los desarrolladores. **Las APIs son los bloques con los que se construyen aplicaciones complejas de nivel empresarial.**

<h2>Brechas de Seguridad Recientes a través de APIs</h2>
- **LinkedIn:** en Junio de 2021, los datos de más de 700 millones de usuarios de LinkedIn fueron puestos en venta en un foro de la dark web, los cuales fueron recopilados mediante la API de LinkedIn. El hacker publicó un registro de 1 millón para confirmar la legitimidad del ataque.
- **Twitter (X):** En Junio de 2022, la información de más de 5,4 millones de usuarios de Twitter fue puesta a la venta en la dark web. Los hacker explotaron un zero-day en la API de Twitter.
- **PIXLR:** En Enero de 2021, PXLR sufrió una brecha de datos que impactó a 1,9 millones de usuarios. Toda la información fue dumpeada por los hackers en un foro de la dark web.
