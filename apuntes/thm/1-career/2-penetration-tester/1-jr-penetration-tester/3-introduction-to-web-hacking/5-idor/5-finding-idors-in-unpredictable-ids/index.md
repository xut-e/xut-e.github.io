<h2>IDs Impredecibles</h2>
Si no se pueden detectar usando los métodos anteriores, un método excelente de detección de IDORs es crear dos cuentas y cambiar los números de ID entre ellas. Si puedes ver el contenido del otro usuario cambiando el ID, encontraste una vulnerabilidad IDOR.
