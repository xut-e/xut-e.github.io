Las soluciones de seguridad de red podrían ser de software o de hardware usadas para monitorizar, detectar y prevenir actividades maliciosas en la red. Se enfoca en proteger a clientes y dispositivos conectados a la red de cooperación. La solución de seguridad de red incluye, pero no está limitada a:

- Firewall de red
- SIEM
- IDS/IPS

------------------------------------------
<h2>Firewall de Red</h2>
Un firewall es el primer punto de comprobación de tráfico no confiable que llega a la red. El firewall filtra el tráfico no confiable antes de pasarlo a la red basado en reglas y políticas. Además, los Firewalls pueden ser usados para separar redes de fuentes de tráfico externo, fuentes de tráfico interno o incluso aplicaciones específicas. Algunos de los firewalls que usan las empresas son:

- Firewalls de filtrado de paquetes
- Firewalls proxy
- Firewalls NAT
- Firewalls de aplicaciones web

-----------------------------------------------
<h2>Security Information and Event Management (SIEM)</h2>
El SIEM combina Security Information Management (SIM) y Security Event Management (SEM) para monitorizar y analizar eventos y rastrear y registrar la información en tiempo real. El SIEM ayuda a los administradoores y blue teamers a monitorizar y rastrear amenazas y vulnerabilidades potenciales antes de que causen daño.

Las soluciones SIEM trabajan como centros de agregación de información donde recolectan archivos de registro de sensores y realizazn funciones en la información recolectada para identificar y detectar amenazas de seguridad o ataques. Algunas de las funciones que ofrece un SIEM son:

- **Gestión de registros:** Captura y recopila información para la red de la empresa a tiempo real.
- **Analítica de eventos:** Aplica analíticas avanzadas para detectar patrones anormales, disponible en el dashboard con tablas y estadísticas.
- **Monitorización de incidentes y alertas de seguridad:** Monitoriza la red entea, incluyendo usuarios conectados, dispositivos, aplicaciones y más. Y tan pronto como los ataques son detectados, avisa a los administradores para que tomen las medidas pertinentes para mitigar.
- **Gestión de leyes y reporte:** Genera reportes en tiempo real en cualquier momento.

El SIEM es capaz de detectar amenazas avanzadas y desconocidas usando inteligencia de amenazas integrada y tecnología IA.

Algunos de los SIEMs más conocidos son:

- Splunk
- LogRhythm NextGen SIEM Platform
- SolarWinds Security Event Manager
- Datadog Security Monitoring
- muchos más...

---------------------------------------------
<h2>Intrusion Detection System and Intrusion Prevention System (NIDS/NIPS)</h2>
Los IDSs/IPSs basados en red tienen un concepto similar al basado en host. La principal diferencia radica en que se centra en la seguridad de la red en lugar de la del host. La solución basada en red está fundamentada por sensores y agentes distribuídos en dispositivos de red y en hosts para recolectar información. La principal diferencia entre un IDS y un IPS es que el IDS requiere de interacción de humanos o software de terceros mientras que el IPS es un sistema de control que acepta o rechaza paquetes basado en políticas y reglas.

Los siguientes son algunos IPSs/IDSs:

- Palo Alto Networks
- Cisco's Next-Generation
- McAfee Network Security Platform (NSP)
- Trend Micro TippingPoint
- Suricata
