Una campaña dirigida de threat intel, tomará todo el conocimiento y tópicos previamente cubiertos y los combinará para crear una campaña bien planificada.

El flujo de tareas en esta unidad sigue el mismo camino que tomarías como red team comenzando a planificar la campaña.

1. Identificar framework y cyber kill chain general.
2. Determinar el adversario objetivo.
3. Identificar los TTPs e IOCs de un adversario.
4. Mapear la threat intelligence obtenida a una cyber kill chain o framework.
5. Esbozar o mantener documentación de engagement.
6. Determinar y usar recursos necesarios del engagement

En esta tarea, estaremos viendo alguno de los procesos de pensamiento del red team, de principio a fin de la planificación de una campaña dirigida de threat intel.

La parte más difícil de la planificación es mapear un par de frameworks diferentes. Para hacer este proceso más sencillo, hemos dado comparación básica de tablas de la cyber kill chain Lockheed Martin y el framework MITRE ATT&CK.

| **Cyber Kill Chain**  | **ATT&CK**            |
| --------------------- | --------------------- |
| Recon                 | Reconnaissance        |
| Weaponization         | Execution             |
| Delivery              | Initial Access        |
| Exploitation          | Initial Access        |
| Installation          | / Defense Evasion     |
| Command & Control     | Command and Control   |
| Actions on Objectives | Exfiltration / Impact |
