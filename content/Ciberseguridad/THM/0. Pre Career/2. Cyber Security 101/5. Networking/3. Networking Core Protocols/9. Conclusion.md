<h2>Puertos Interesantes</h2>

| **Protocolo** | **Transport Protocol** | **Default Port Number** |
| ------------- | ---------------------- | ----------------------- |
| TELNET        | TCP                    | 23                      |
| DNS           | UDP or TCP             | 53                      |
| HTTP          | TCP                    | 80                      |
| HTTPS         | TCP                    | 443                     |
| FTP           | TCP                    | 21                      |
| SMTP          | TCP                    | 25                      |
| POP3          | TCP                    | 110                     |
| IMAP          | TCP                    | 143                     |
