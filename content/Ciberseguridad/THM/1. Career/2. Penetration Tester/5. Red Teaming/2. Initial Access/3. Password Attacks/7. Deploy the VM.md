Despliega la máquina y aplica el conocimiento adquirido. La VM tiene varios servicios online en los que realizar ataques de contraseñas. Se requieren wordlists personalizadas.

Recomendamos el uso de `https://clinic.thmredteam.com` para crear dicha wordlist. Para generarla usando `cewl` contra la página:

```bash
cewl -m 8 -w clinic.lst https://clinic.thmredteam.com
```

>[!NOTE] Ten en cuenta que también tendrás que generar una wordlist de nombre de usuario como se muestra en la tarea 3 para las preguntas de ataque online.
