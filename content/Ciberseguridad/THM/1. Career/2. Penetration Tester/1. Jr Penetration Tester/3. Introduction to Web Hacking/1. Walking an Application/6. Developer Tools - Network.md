La pestaña de red en las herramientas de desarrollador puede usarse para llevar una cuenta de cada petición externa que hace la página.

Prueba a hacer esto en la página de `/contact`. Puedes borrar la lista con el botón de la basura, por si se sobrepobla.

Con la pestaña de red abierta, prueba a rellenar la información de contacto y presionar **Send Message**. Notarás un evento, esto es la solicitud siendo subida usando un método llamado AJAX. AJAX es un método para mandar y recibir información por red en una aplicación web sin interferir cambiando la web actual.
