En esta unidad hemos cubierto tres tipos de escaneo:

| Tipo de Escaneo de Puertos | Comando de Ejemplo           |
| -------------------------- | ---------------------------- |
| TCP Connect Scan           | `nmap -sT 10.82.141.81`      |
| TCP SYN Scan               | `sudo nmap -sS 10.82.141.81` |
| UDP Scan                   | `sudo nmap -sU 10.82.141.81` |
Además hemos visto las siguientes opciones para refinarlos:

| Opción                  | Propósito                                 |
| ----------------------- | ----------------------------------------- |
| `-p-`                   | Todos los puertos                         |
| `-p1-1023`              | Puertos del 1 al 1023                     |
| `-F`                    | Los 100 puertos más comunes               |
| `-r`                    | Escanear los puertos en orden consecutivo |
| `-T<0-5>`               | Rapidez del escaneo                       |
| `--max-rate 50`         | Cantidad máxima de paquetes por segundo   |
| `--min-rate 15`         | Cantidad mínima de paquetes por segundo   |
| `--min-parallelism 100` | Cantidad de hilos escaneando              |
