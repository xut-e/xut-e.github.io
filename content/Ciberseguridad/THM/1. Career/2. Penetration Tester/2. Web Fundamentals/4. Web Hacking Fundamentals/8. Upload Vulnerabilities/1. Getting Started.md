Esta tarea te enseñará a configurar el [archivo de hosts](https://www.ionos.co.uk/digitalguide/server/configuration/hosts-file/) de tu dispositivo. El archivo hosts se usa para mapear, bypasseando el DNS. Es decir, permite relacionar un dominio con una IP sin necesidad de pasar por un DNS. Es extremadamente útil en entornos donde el DNS no está disponible, como TryHackMe.

------------------------------------------
Primero desplegamos la máquina y le damos unos 5 minutos para arrancar.

Ahora usando una sesión administrativa y tu editor de texto favorito, abre el archivo `hosts` de tu equipo:

- En Linux y MacOS se encuentra en `/etc/hosts`.
- En Windows está en `C:\Windows\System32\drivers\etc\hosts`.

Añade la siguiente línea (en singular) al final del archivo.

`10.80.160.47    overwrite.uploadvulns.thm shell.uploadvulns.thm java.uploadvulns.thm annex.uploadvulns.thm magic.uploadvulns.thm jewel.uploadvulns.thm demo.uploadvulns.thm`

La sintaxis es `IP + TAB + Dominio` --> `192.168.2.1    dominio.com`.

>[!IMPORTANT] Cuando termines de usar la máquina asegúrate de eliminar la línea.
