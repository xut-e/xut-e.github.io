<h2>Bloques de Cifrado</h2>
AES requiere que el input para la encriptación sea de tamaño fijo en bloques para cada ronda de encriptación. Esto es por lo que es referido como block cipher. El único problema es que raramente tienes un mensaje que encaje perfectamente en los bloques requeridos por AES. Es por esto por lo que los mensajes son separados por bloques y para el último bloque se usa el padding. Aprenderás más sobre el padding y sus vulnerabilidades en la siguiente unidad: **Padding Oracles**. Es por esto por lo que se ha simplificado el padding en esta unidad para rellenar con `*` cuando no se alcance el tamaño requerido.

```python
DEFAULT_PADDING_SYMBOL = "*"
def custom_pad(message):
    while len(message) % DEFAULT_BLOCK_SIZE != 0:
        message += DEFAULT_PADDING_SYMBOL
    return message
def custom_unpad(message):
    count  = 0
    for x in range (1, len(message)):
        if message[len(message) - x] == DEFAULT_PADDING_SYMBOL:
            count += 1
        else:
            break
    print (count)
    message = message[0:len(message)-count]
    return message
```

AES requiere que el mensaje sea partido en bloques de 16 bytes. Al trabajar con texto crudo, un carácter ASCII es 1 byte. En fotos en blanco y negro, cada píxel es también un byte, pero para imágenes en color pueden ser 3 bytes o más dependiendo de la calidad.

<h2>Modos de Cifrado</h2>
Si tenemos todos estos bloques, la pregunta es cómo deberíamos encadenarlos para crear el ciphertext final. El método de encadenado es llamado Cipher Block Mode. Hay varios modos que pueden ser usados. En esta room, nos centraremos en un modo inseguro llamado Electronic CodeBook (ECB). Simplemente, ECB no encadena los bloques sino que encripta cada uno por separado:

<h3>Paso 3: Crear el Output Final</h3>
El output final para ECB es creado simplemente añadiendo bloques juntos, resultando en el ciphertext final:

```hex
Ciphertext: 92 FF C7 FE CD 0D 04 13 E8 B2 63 6D F4 38 BF 2A FE 7C 80 45 6D 0A 87 C7 7A 20 61 78 B9 7A 19 33
```

En Python, el módulo `Crypto.Cipher.AES` de la librería `pycryptodome` puede ser usado para la encriptación con AES. Usaremos `AES.MODE_ECB` para la encriptación como se muestra debajo:

```python
plaintext_bytes = custom_pad(plaintext).encode()  
cipher = AES.new(key.encode(), AES.MODE_ECB)  
encrypted_bytes = cipher.encrypt(plaintext_bytes)  
encrypted_message = binascii.hexlify(encrypted_bytes).decode()
```
