<h2>Rust-Dance: The Virus That Dances</h2>

<h3>The Story</h3>
A new threat has emerged from the depths of the network. They call it **Rust-Dance** - a virus that doesn't just infect systems, it makes them _dance_.

Every file, every process, every byte... now moves to its rhythm.

Your mission: Find the unlock code that stops the dance. Reverse engineer the virus, understand its choreography, and break free.

The flag is definitely not "unlockcode123456789", so don't even try.

The clock is ticking. The dance continues...

----------------------
<h3>How to Run</h3>

<h4>Requirements</h4>

You may use one of the following:

- Windows Subsystem for Linux (WSL)
- Linux x86_64 system (native or VM)

<h4>Execution</h4>

```bash
# Make the binary executable (if needed)
chmod +x rust-dance

# Run the challenge
./rust-dance

# The virus will prompt you for the unlock code
# Your goal: find the correct code to reveal the flag
```

<h3>Rules</h3>

1. The flag format is: `MWC{...}`
2. You must find the unlock code through reverse engineering
3. Standard reversing tools are allowed (IDA, Ghidra, radare2, GDB, etc.)
4. Have fun and learn!

<h3>Flag Submission</h3>

Once you find the flag, submit it on the CTF platform in the format:

```
MWC{hash}
```

------------------------------

Good luck, reverser. Stop the dance.
