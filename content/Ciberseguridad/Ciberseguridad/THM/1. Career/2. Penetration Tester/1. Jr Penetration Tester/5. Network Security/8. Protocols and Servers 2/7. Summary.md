En esta unidad hemos visto varios protocolos, su uso, ataques contra ellos y medidas de mitigación.

| Protocolo | Puerto TCP | Aplicación(es)                  | Seguridad de la Información |
| --------- | ---------- | ------------------------------- | --------------------------- |
| FTP       | 21         | File Transfer                   | Cleartext                   |
| FTPS      | 990        | File Transfer                   | Encrypted                   |
| HTTP      | 80         | Worldwide Web                   | Cleartext                   |
| HTTPS     | 443        | Worldwide Web                   | Encrypted                   |
| IMAP      | 143        | Email (MDA)                     | Cleartext                   |
| IMAPS     | 993        | Email (MDA)                     | Encrypted                   |
| POP3      | 110        | Email (MDA)                     | Cleartext                   |
| POP3S     | 995        | Email (MDA)                     | Encrypted                   |
| SFTP      | 22         | File Transfer                   | Encrypted                   |
| SSH       | 22         | Remote Access and File Transfer | Encrypted                   |
| SMTP      | 25         | Email (MTA)                     | Cleartext                   |
| SMTPS     | 465        | Email (MTA)                     | Encrypted                   |
| Telnet    | 23         | Remote Access                   | Cleartext                   |
Además hemos visto una herramienta para realizar ataques de contraseñas, Hydra.

| Opción            | Explicación                                                       |
| ----------------- | ----------------------------------------------------------------- |
| `-l username`     | Ofrece el nombre de usuario                                       |
| `-P WordList.txt` | Especifica el diccionario a usar                                  |
| `server service`  | Configura la dirección del servidor y el servicio a atacar.       |
| `-s PORT`         | Usar en caso de que el servicio no esté en su puerto por defecto. |
| `-V` or `-vV`     | Muestra las combinaciones que se están usando.                    |
| `-d`              | Muestra la depuración si verbose no ayuda.                        |
