La fuente web es el código legible por humanos devuelto al navegador cada vez que hacemos una petición.

El código devuelto está hecho de HTML, CSS y JavaScript y es lo que le dice al navegador qué contenido mostrar, cómo mostrarlo y añade elementos de interacción.

------------------------------
<h2>¿Cómo vemos el código fuente?</h2>
1. Al ver una página web podemos hacer click derecho en la página y veremos una opción que dice ver código fuente.
2. La mayoría de lso navegadores soportan el método **view-sorce** enn frente de la URL, por ejemplo: "view-source:https://www.google.com".
3. En el menú del buscador encontrarás una opción para ver el código fuente. A veces puede estar en submenús como en opciones de desarrollador.
