La interfaz de extensiones en Burp Suite ofrece una vista de las extensiones cargada en la herramienta. Echemos un vistazo a los diferentes componentes.

![[Pasted image 20251117092220.png]]

1. **Extensions List:** La caja de arriba muestra una lista de las extensiones instaladas actualmente para el proyecto actual. Permite activar o desactiver extensiones.
2. **Managing Extensions:** En la parte izquierda de la interfaz, hay opciones para manejar extensiones:
	- **Add:** Puedes usar este botón para instalar nuevas extensiones de archivos en tu disco. Pueden ser programados por ti u obtenidos de las fuentes externas.
	- **Remove:** Este botón permite desinstalar extensiones.
	- **Up/Down:** Estos botones controlan el orden en el que se listan las extensiones instaladas. Es importante porque las extensiones se aplican de arriba a abajo y el orden importa.
3. **Details, Output and Errors:** Hacia la parte inferior de la ventana hay secciones para la extensión seleccionada actualmente:
	- **Details:** Esta sección ofrece información sobre la extensión seleccionada, como el nombre, versión y descripción.
	- **Output:** Las extensiones pueden producir un output durante su ejecución, y esta sección muestra cualquier output o resultados relevantes.
	- **Errors:** Si una extensión se encuentra algún error durante su ejecución, serán mostradas aquí.
