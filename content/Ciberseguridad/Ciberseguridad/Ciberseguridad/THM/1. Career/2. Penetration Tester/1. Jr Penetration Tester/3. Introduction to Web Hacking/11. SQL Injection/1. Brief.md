SQL Injection, más conocida como SQLi, es un ataque en el servidor de base de datos de una aplicación web que causa que se ejecuten queries maliciosas. Cuando una aplicación web se comunica con una base de datos usando un input del usuario que no ha sido correctamente sanitizado, ahí yace el potencial del atacante para robar, borrar o alterar información privada y de clientes. Por esto la SQLi es una de las vulnerabilidades más viejas que existen (y también podría ser una de las más dañinas).

En esta unidad aprenderemos lo que son las bases de datos, qué es SQL con algunos comandos SQL básicos, cómo detectar vulnerabilidades SQL, cómo explotarlas y, como desarrollador, cómo protegernos de estas.
