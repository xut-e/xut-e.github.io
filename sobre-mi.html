Sobre mí

Soy xut, especialista en ciberseguridad y estudiante en la Technische Hochschule Mittelhessen (THM). Mi objetivo es compartir conocimiento técnico en castellano, documentando mi proceso de aprendizaje en hacking ético, redes, sistemas y seguridad ofensiva.

🧠 Qué encontrarás aquí

Este sitio es mi base de conocimiento personal, donde recopilo apuntes de TryHackMe, guías, notas y documentación técnica. Todo está pensado para ser útil, limpio y accesible tanto para mí como para cualquiera que esté aprendiendo ciberseguridad desde cero.

Con el tiempo, este espacio evolucionará hacia un portfolio técnico que refleje mis proyectos, herramientas y certificaciones.

💻 Tecnologías y enfoque

  • Linux, bash scripting, automatización y seguridad de sistemas.
  • Análisis de vulnerabilidades, pentesting y explotación.
  • Documentación técnica con Obsidian, Git y GitHub Pages.
  • Ética, aprendizaje continuo y curiosidad como principios base.
